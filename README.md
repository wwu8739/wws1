The Foodie Web App is a clean, modern single-page front-end food ordering demo built with HTML, CSS, JavaScript, and Bootstrap. Users can browse a menu of dishes loaded from a local JSON file, search by name or category, sort items by price or name, add selections to a real-time dynamic cart using sessionStorage, and view an updating order summary—all without any backend. It's fully responsive, mobile-first, and includes a simple login for session persistence.
